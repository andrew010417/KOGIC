# ONT QC using NanoPlot

## Description
Perform quality control for ONT ultra-long reads using NanoPlot.

## Input
- FASTQ file (ONT raw reads)

## Output
- NanoPlot HTML report
- Read length distribution
- N50, Yield

## Usage
qsub nanoplot.sh

## Notes
- NanoPlot installed via pip
- PATH manually added in script
